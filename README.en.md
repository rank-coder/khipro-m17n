# Documentation

## Introduction
While Avro is popular for Bangla phonetic typing, frequent use of the Shift key and distance of certain keys (e.g., ^ for ‡¶Å) can disrupt typing speed and flow. This makes it difficult to increase speed beyond a certain limit.

To overcome these limitations, we developed the Khipro layout, inspired by China's Pinyin system. Because Pinyin is case-insensitive and features intelligent suggestions, Chinese users can often type in Chinese faster than English.

Khipro brings that same concept‚ÄîBangla can be written quickly and smoothly without using Shift. All necessary characters can be typed without moving your hands away from the main keyboard area.

On Linux, Khipro becomes even faster through Typing Booster, providing suggestions for multiple upcoming words. On Windows, Android, and other platforms, predictive text features can be used with apps that implement Khipro.

In Khipro, it's possible to force a vowel sign (*kar*) to become a vowel or vice versa with just a single button press. There are many more such features.
